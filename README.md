# K-pop女爱豆风格测试

一个基于H5的互动测试小程序，用户通过回答10道问题，匹配最适合自己参考的女爱豆风格。

## 📋 项目简介

本项目是一个响应式H5页面，设计用于在手机上流畅运行。用户通过回答关于身高、脸型、肤色、五官量感、眼睛形状、嘴唇厚度、适合发色、头发长短、身材和气质风格的问题，系统会根据加权算法匹配最相似的女爱豆，并展示匹配度最高的5位爱豆。

## ✨ 功能特点

- **响应式设计**：适配手机、平板等不同设备屏幕
- **互动测试**：10道选择题，最后一题支持最多3项多选
- **智能匹配**：基于加权算法计算匹配度，风格匹配优先
- **视觉效果**：渐变背景、动画过渡、进度条展示
- **结果展示**：显示匹配度最高的5位爱豆，包括匹配百分比
- **版权信息**：首页显示原创作者信息

## 🚀 如何使用

### 本地访问
1. 克隆或下载本项目到本地
2. 直接打开 `kpop-test.html` 文件即可开始测试

### GitHub Pages部署
1. 将项目推送到GitHub仓库
2. 进入仓库 **Settings** → **Pages**
3. 选择部署分支（如 `main`）和文件夹（如 `/(root)`）
4. 等待1-2分钟后，访问生成的GitHub Pages URL

## 📝 测试流程

1. **开始页面**：点击「开始测试」按钮进入测试
2. **回答问题**：依次回答10道问题，最后一题可选择1-3个选项
3. **查看结果**：测试完成后显示匹配度最高的5位女爱豆
4. **重新测试**：点击「重新测试」按钮可再次开始

## 🎯 匹配算法

- **权重分配**：
  - 身高、眼睛形状、嘴唇厚度、适合发色：每题10分
  - 脸型、五官量感、气质风格：每题15分
  - 肤色、头发长短、身材：每题5分
- **风格优先**：总分相同时，以风格匹配得分最高的优先
- **多选题计算**：根据匹配选项数量按比例计算得分
